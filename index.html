Project 1

belajar bordir dasar

Deskripsi Proyek:

Proyek ini merupakan inisiatif pribadi untuk mempelajari teknik bordir tangan sebagai hobi sekaligus menambah keterampilan kerajinan. Sebagai pemula, saya fokus memahami langkah-langkah dasar—mulai dari cara memegang jarum, memilih jenis kain dan benang, hingga membuat pola bordir sederhana. Selama proses, saya mencoba beberapa motif sederhana (misalnya daun, bunga kecil, dan pola geometris dasar) untuk melatih ketelitian dan kestabilan jahitan.

Langkah Pembelajaran:

1. Latihan Dasar Jahitan

  • Mempraktikkan backstitch pada garis lurus dan lengkung untuk mengenali kontrol jari.
  • Mencoba satin stitch (jahitan datar) pada bentuk oval kecil.
  • Belajar french knot untuk membuat titik-titik dekoratif (misalnya stamen bunga).

    • 2. Membuat Pola Sederhana:

  • Menggambar satu atau dua tipe motif: daun tunggal, bunga lima kelopak, dan pola geometris (lingkaran, segitiga).
  • Mengaplikasikan pola pada kain secara bertahap, memulai dari satu motif lalu menggabungkan dua motif berbeda pada satu kain.

    • Project 2

    Belajar Dasar Website (HTML, CSS, JavaScript)

    Tujuan:

    Mempelajari struktur dan sintaks dasar HTML untuk membuat kerangka halaman, menambahkan gaya dengan CSS, serta menambahkan interaktivitas sederhana menggunakan JavaScript.

    langkah utama:

    1. Persiapan Struktur HTML

  • Membuat file index.html dengan elemen dasar: DOCTYPE html, head, body.
  • Menambahkan konten sederhana: judul halaman, paragraf perkenalan, tombol, dan form input teks.

    • 2. Menambahkan Gaya dengan CSS

  • Membuat file styles.css dan menautkan ke index.html.
  • Mengatur layout menggunakan flexbox atau grid ringan (misalnya header dan konten utama).
  • Menyesuaikan warna, ukuran font, spasi (margin/padding), dan menambahkan hover effect pada tombol.
  • Menggunakan dasar DOM manipulation: document.getElementById() atau querySelector().

    • 3. Interaktivitas dengan JavaScript

  • Membuat file script.js dan menautkan ke index.html di bagian bawah ""body"".
  • Menambahkan event listener pada tombol, misalnya saat diklik, muncul alert atau mengubah teks di halaman.
  • Mengambil nilai dari input teks, lalu menampilkan hasilnya di elemen "div" sebagai respons dinamis

    • 4. Testing & Debugging

  • Mencoba membuka halaman di browser, inspeksi elemen, dan memperbaiki kesalahan di konsol.
  • Memastikan CSS dan JavaScript terhubung dengan benar (cek Network dan Console).

    • Hasil & Pembelajaran:

  • HTML: Memahami struktur dasar dokumen dan tag-tag penting (heading, paragraf, list, form).
  • CSS: Dapat membuat layout sederhana, menyesuaikan tampilan, dan membuat halaman responsif untuk layar kecil.
  • JavaScript: Mampu menambahkan event listener, memanipulasi konten DOM, dan membuat interaksi dasar (alert, ubah teks).
